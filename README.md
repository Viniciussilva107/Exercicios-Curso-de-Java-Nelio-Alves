# ☕ Curso de Java - Nelio Alves

Este repositório contém todos os meus projetos, exercícios e anotações desenvolvidos durante o Curso Completo de Java, ministrado pelo professor Nelio Alves.

O objetivo deste repositório é documentar meu progresso e consolidar os conceitos de Programação Orientada a Objetos (POO), estruturas de dados e boas práticas de desenvolvimento em Java, conectando a base teórica com a aplicação prática.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **Versionamento:** Git e GitHub
